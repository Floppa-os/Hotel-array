# Новое Вельяминово
Это обычный посёлок
# Програма
## Запуск
Для запуска откройте `main.py` командой `python main.py`
## Параметры
Параметры находятся в `config.py`
## Компиляция
### Подготовка инструментов
`pip install pyinstaller`
### Компиляция
`pyinstaller main.py --onefile`
# Список команд
## save
Позволяет сохранить параметры
## load
Позволяет загрузить параметры
## add rooms_number_user
Эта команда добавляет права на комнату указанному человеку.
Надо заменить number на номер комнаты
## del rooms_number_user
Эта команда удаляет права на комнату последнему человеку в списке.
Надо заменить number на номер комнаты
## reset >
Эта команда сбрасывает параметры
